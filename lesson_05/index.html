CSS position

  • Position: fixed - вырывает элемент из потока и делает его невидимым для остальных элементов.
  • Position: sticky - приклеивает элемент к месту, сохраняя за собой место, после выравнивания из потока (идеально для header). Чтобы stycky сработал - нужно не забыть задать значение по одной из осей.
  • Position: absolute - двигает элемент по осям относительно body (по умолчанию) или первого спозиционированного родителя.
  • Position: relative - не вырывает элемент из потока, что идеально подходит для позиционирования родителя для absolute - теперь не придется двигать элемент относительно body.
  • Absolute - указываем для ребенка, элемента который будем двигать.
  • Relativ - для элемента родителя, чтобы двигать ребенка относительно него.

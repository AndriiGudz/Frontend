Flexbox

Гибкий технология позиционирования элементов в CSS

  • Чтобы включить отображение flex в контейнере нужно задать свойство для родителя.
  • justify-content: - задает правила на главной оси
  • align-items: - задает правила на поперечной оси
  • Align-self - вы можете применить к элементу ребенку, чтобы изменить его положение по поперечной оси.
